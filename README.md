# Ostergeschichte in Echtzeit

Erlebe die Ereignisse der Karwoche in Echtzeit - von Gründonnerstag bis Ostersonntag.

## Features

- 📱 WhatsApp-Kanal für Benachrichtigungen
- ✈️ Telegram-Bot mit automatischen Nachrichten
- 🌐 Timeline-Website zum Durchstöbern
- 🕐 Ereignisse zu historisch korrekten Zeitpunkten

## Deployment

Diese Website ist für Vercel optimiert:

1. Repository auf GitHub pushen
2. Mit Vercel verbinden
3. Automatisches Deployment

## Struktur

```
/
├── index.html          # Landing-Page
├── style.css           # Styling

├── datenschutz.html    # Datenschutz
├── timeline/           # Timeline-App

```

- Vanilla HTML/CSS/JavaScript
